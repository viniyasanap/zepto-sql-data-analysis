# 🛒 Zepto SQL Data Analysis

## 📖 Overview

This project focuses on analyzing the Zepto grocery products dataset using PostgreSQL. The dataset was imported from a CSV file, cleaned, and explored using SQL to answer business-related questions and generate meaningful insights.

The project demonstrates practical SQL skills commonly used in data analytics, including data filtering, aggregation, sorting, grouping, and business analysis.

---

## 🎯 Project Objectives

- Import and manage a real-world dataset in PostgreSQL.
- Perform data cleaning and validation.
- Analyze product pricing, discounts, and stock availability.
- Solve business problems using SQL queries.
- Generate insights to support data-driven decision-making.

---

## 📂 Dataset

The project uses the *Zepto Grocery Products Dataset*, which contains information about products available on the Zepto platform.

### Features include:
- Product Name
- Category
- MRP
- Discount Percentage
- Discount Selling Price
- Available Quantity
- Weight (in grams)
- Stock Availability

## 📊 Analysis Performed

The project includes SQL queries to:

- Count total products
- Identify products with the highest discounts
- Find high-priced products that are out of stock
- Calculate estimated revenue by category
- Compute average discounts for each category
- Calculate price per gram for products
- Filter and sort products based on business conditions
- Perform aggregations using COUNT(), SUM(), AVG(), GROUP BY, and ORDER BY

## ▶️ How to Run

1. Clone this repository.
2. Open PostgreSQL and pgAdmin.
3. Create a new database.
4. Import the zepto_v2.csv dataset.
5. Open the zepto_analysis.sql file in the Query Tool.
6. Execute the SQL queries to reproduce the analysis.
